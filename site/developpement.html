Bibliothèques

  • networkx : Permet de manipuler les graphes.
    • Graph() : Initialise le graphe sans noeuds et arêtes
    • gnm_random_graph() : On peut créer des graphes aléatoires selon le nombre de noeuds et d'arêtes
    • relabel_nodes() : On renomme les noeuds. Celà nous permet de passer des entiers en chaîne de caractères puis travailler avec les noeuds plus facilement
    • random_layout() : Positionne aléatoirement les noeuds dans l'espace
    • draw_networkx() : On peut dessiner le graphe avec les caractéristiques que l'on souhaite (couleur des noeuds, taille de la police,avec ou sans label...)
  • matplotlib.pyplot : Permet d'afficher des graphiques.
    • subplot() : Fractionne la fenêtre
    • title() : Met un titre au graphe
    • show() : Affiche les graphes
  • pysat : Permet d'étudier les problèmes de satisfaction SAT
    • formula : Un module de pysat pour travailler facilement avec des formules booléennes
      • CNF() : Permet d'étudier des instances CNF
    • solvers : Un module de pysat pour accéder à différents solveur de problème SAT
      • Solver() : Nous permet de voir si la formule boolénnes est satisfiable
  • pycsp3 : Permet de mettre des contraintes sur des variables
    • VarArray() : Structure de données à la quelle on dit la taille et le domaine de définition des valeurs de la structure
    • satisfy() : Permet d'entrer toutes les contraintes que l'on souhaite pour la structure
    • solve() : Vérifie si la structure de donnée est satisfiable par rapport aux contraines données dans satisfy()
    • values() : Pour obtenir les valeurs de la base de donnée
